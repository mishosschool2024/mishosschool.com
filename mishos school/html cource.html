Lesson 1: Introduction to HTML

HTML (Hypertext Markup Language) is a markup language used for creating websites. With HTML, you can structure and format text, images, and other elements. HTML is the backbone of a website, defining its structure and content.

What is HTML?

HTML is the standard language used for creating and managing websites. Editing HTML requires creating formatted content and visual design, which is important for creating websites that are easy to understand and appealing to users.

Structure of HTML

An HTML document begins with <!DOCTYPE html> and then the <html> tag, indicating the version of HTML. An HTML page must include <head> and <body> sections. The <head> contains information about the page, while the <body> is the part that appears on the user's screen.

<!DOCTYPE html>
<html>
    <head>
        <title>My First Website</title>
    </head>
    <body>
        <h1>Hello World</h1>
        <p>This is my first HTML page!</p>
    </body>
</html>

Task

Create a simple HTML file containing text content and save it as .html. Use the above example to create your own HTML page.

Lesson 2: HTML Elements

HTML elements are the basic components used to create websites. Each element contains opening and closing tags, for example, <p> and </p>.

Types of Elements

  • Block Elements: These elements start on a new line and occupy wide spaces. For example, <div>, <p>, <h1>.
  • Inline Elements: These elements do not start on a new line and are used within other elements. For example, <span>, <a>.

Example

Here is a simple example of how to use HTML elements:

<p>This is text inside a block element</strong></p>

Lesson 3: HTML Attributes

HTML attributes are added to elements to provide additional information. Adding attributes allows users to add more details, such as image settings, links to other websites, and more.

Using Attributes

<a href="https://www.example.com">This is a link</a>

Example

Example of adding attributes:

<img src="image.jpg" alt="This is an image">

Lesson 4: HTML Forms

HTML forms allow for the collection of user information. Forms are mainly used to collect user information, such as name, email, and more. A form can contain several elements, such as text fields, radio buttons, and buttons.

Form Structure

<form action="url" method="GET/POST">
    <input type="text" name="username">
    <input type="submit" value="Submit">
</form>

Lesson 5: HTML Images

To add images in HTML, the <img> element is used. Adding images to a website significantly enhances the visual aspect and can include additional content.

Image Attributes

Attributes for adding images:

  • src: Specifies the source of the image.
  • alt: Contains a text description that appears instead of the image if the image does not load.

Example

<img src="image.jpg" alt="This is an image">

Task

Lesson 6: HTML Links

To add links in HTML, the <a> element is used. Links allow users to navigate to other pages or websites. The href attribute within the link element specifies the link's target.

Using Links

<a href="https://www.example.com">This is a link</a>

Lesson 7: HTML Sections

HTML allows for organizing the content into sections using the <div> and <span> elements.

Div and Span Elements

The <div> element is a block-level element used for grouping content, and the <span> element is an inline element used for grouping text.

<div>
    <p>This is a paragraph inside a div element.</p>
</div>

Task

Use div and span elements to organize your HTML content into different sections.

Lesson 8: HTML Tables

HTML tables allow for organizing and displaying data in rows and columns. Using the <table>, <tr>, <th>, and <td> elements, you can create tables.

Table Structure

<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
</table>

Task

Lesson 9: HTML Lists

HTML lists allow for organizing content into ordered and unordered lists. The <ul> element is used for unordered lists, and the <ol> element is used for ordered lists.

Creating Lists

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

Lesson 10: HTML Iframes

The <iframe> element is used to embed another HTML page within the current page. Iframes are useful for embedding external content, such as videos or interactive maps.

Using Iframes

<iframe src="https://www.example.com" width="600" height="400"></iframe>
