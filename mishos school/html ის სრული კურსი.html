HTML გაკვეთილები

გაკვეთილი 1: HTML-ის შესავალი

HTML (Hypertext Markup Language) არის მარკაპის ენა, რომელიც გამოიყენება ვებსაიტების შექმნისთვის. HTML-ის მეშვეობით შეგიძლიათ მოაწყოთ და ფორმატირება გაუკეთოთ ტექსტს, სურათებს და სხვა ელემენტებს. HTML არის ვებსაიტის რბილული, რომელიც განსაზღვრავს მის სტრუქტურას და შინაარსს.

რა არის HTML?

HTML არის სტანდარტული ენა, რომლის მეშვეობითაც შეიძლება ვებსაიტების შექმნა და მართვა. HTML-ის გასწორება მოითხოვს ფორმატირებული შინაარსის და ვიზუალური დიზაინის შექმნას, რაც მნიშვნელოვანია მომხმარებლებისთვის მარტივად გასაგები და მიმზიდველი ვებსაიტების შექმნისათვის.

HTML-ის სტრუქტურა

HTML დოკუმენტი იწყება <!DOCTYPE html> და შემდეგ <html> ტეგით, რაც აღნიშნავს HTML-ის ვერსიას. HTML-ის გვერდი უნდა შეიცავდეს <head> და <body> ნაწილებს. <head> შეიცავს ინფორმაციას გვერდზე, ხოლო <body> არის ის ნაწილი, რომელიც გამოჩნდება მომხმარებლის ეკრანზე.

<!DOCTYPE html>
<html>
    <head>
        <title>ჩემი პირველი ვებსაიტი</title>
    </head>
    <body>
        <h1>გამარჯობა მსოფლიო</h1>
        <p>ეს არის ჩემი პირველი HTML გვერდი!</p>
    </body>
</html>

დავალება

შექმენით მარტივი HTML ფაილი, რომელიც შეიცავს ტექსტურ შინაარსს და შეინახეთ იგი .html ფორმატში. გამოიყენეთ ზემოთ მოცემული მაგალითი, რათა შექმნათ საკუთარი HTML გვერდი.

გაკვეთილი 2: HTML-ის ელემენტები

HTML-ის ელემენტები არიან ძირითადი კომპონენტები, რომლებიც გამოიყენება ვებსაიტების შექმნაში. თითოეული ელემენტი შეიცავს გახსნის და დახურვის ტეგებს, მაგალითად <p> და </p>.

ელემენტების ტიპები

  • ბლოკური ელემენტები: ეს ელემენტები იწყება ახალ რიგში და ფართო სივრცით იყოფა. მაგალითად, <div>, <p>, <h1>.
  • inline ელემენტები: ეს ელემენტები არ იწყებენ ახალ რიგს და თან მყოფ ელემენტებს შორის ფართო სივრცეში მყოფ ადგილას გამოიყენება. მაგალითად, <span>, <a>.

მაგალითი

აქ არის მარტივი მაგალითი, როგორ უნდა გამოვიყენოთ HTML ელემენტები:

<p>ეს არის ტექსტი, რომელიც მოთავსებულია ბლოკური ელემენტით</strong></p>

დავალება

შექმენით რამდენიმე ბლოკური და inline ელემენტი. დაწერეთ თავისი სახელები და ასაკი HTML-ის გვერდზე, რაც გამოიყენებს <ul> და <li> ელემენტებს.

გაკვეთილი 3: HTML-ის ატრიბუტები

HTML-ის ატრიბუტები ემატება ელემენტებზე, რათა მათ უფრო მეტი ინფორმაცია მიაწვდოს. ატრიბუტების დამატება საშუალებას გაწვდოს მომხმარებლებს მეტი დეტალების, როგორიცაა სურათების პარამეტრები, ვებსაიტზე გადასვლის ბმულები და სხვა.

ატრიბუტების გამოყენება

<a href="https://www.example.com">ეს არის ლინკი</a>

მაგალითი

ატრიბუტების დამატების მაგალითი:

<img src="image.jpg" alt="ეს არის სურათი">

დავალება

დაამატეთ ატრიბუტები თქვენს HTML ელემენტებზე. შექმენით HTML გვერდი, რომელიც შეიცავს სურათებს და ლინკებს.

გაკვეთილი 4: HTML-ის ფორმები

HTML ფორმები საშუალებას გაწვდოს მომხმარებლის ინფორმაციების შეგროვების. ფორმები ძირითადად გამოიყენება მომხმარებლის ინფორმაციების შეგროვებისთვის, როგორიცაა სახელი, ელექტრონული ფოსტა და სხვა. ფორმა შეიძლება შეიცავდეს რამდენიმე ელემენტს, როგორიცაა ტექსტური ველები, რადიო ღილაკები და ღილაკები.

ფორმის სტრუქტურა

<form action="url" method="GET/POST">
    <input type="text" name="username">
    <input type="submit" value="გაგზავნა">
</form>

დავალება

შექმენით ფორმა, რომელიც ითხოვს სახელს, პაროლს და ელექტრონულ ფოსტას. დაამატეთ ველის დატოვების ბმული, რომელიც შეამოწმებს, რომ ინფორმაცია სწორია.

გაკვეთილი 5: HTML-ის სურათები

HTML სურათების დასამატებლად გამოიყენება <img> ელემენტი. სურათების დამატება ვებსაიტზე მნიშვნელოვნად აუმჯობესებს ვიზუალურ მხარეს და შესაძლებელია სხვა შინაარსის დამატება.

სურათის ატრიბუტები

სურათზე დამატების ატრიბუტებია:

  • src: მიუთითებს სურათის წყაროს.
  • alt: შეიცავს ტექსტურ აღწერას, რომელიც გამოჩნდება სურათის ნაცვლად, თუ სურათი არ იტვირთება.

მაგალითი

<img src="image.jpg" alt="ეს არის სურათი">

დავალება

დაამატეთ რამდენიმე სურათი თქვენს HTML გვერდზე. დარწმუნდით, რომ გამოყენებული ატრიბუტები სწორად არიან ჩამოყალიბებული.

გაკვეთილი 6: HTML-ის ლინკები

HTML ლინკების დასამატებლად გამოიყენება <a> ელემენტი. ლინკები საშუალებას გაწვდოს მომხმარებლებს გადადონ სხვა გვერდებზე ან ვებსაიტებზე. ლინკის ელემენტში ატრიბუტები, როგორიცაა href, მიუთითებს ლინკის მიზანს.

ლინკის გამოყენება

<a href="https://www.example.com">ეს არის ლინკი</a>

დავალება

შექმენით რამდენიმე ლინკი, რომლებიც მიუთითებენ სხვადასხვა ვებსაიტებზე. დაამატეთ ლინკები თქვენს მიერ შექმნილ გვერდზე.

გაკვეთილი 7: HTML-ის სექციები

HTML-ის სექციები, როგორიცაა <header>, <nav>, <main>, <footer>, გამოყენებულია შინაარსის ორგანიზებისთვის. ეს ელემენტები განსაზღვრავენ ვებსაიტის სხვადასხვა ნაწილებს.

მაგალითი

<header>
    <h1>ეს არის სათაური</h1>
</header>

დავალება

შექმენით HTML გვერდი, რომელიც იყენებს სექციების ელემენტებს. განსაზღვრეთ <header>, <nav>, <main> და <footer>.

გაკვეთილი 8: HTML-ის პალიტრა

HTML პალიტრა არის ელემენტი, რომელიც გამოიყენება შინაარსის გადანაწილებისთვის. იგი საშუალებას გაწვდოს ჯგუფი სხვადასხვა ელემენტების, როგორიცაა სექციები, სურათები და ტექსტი.

მაგალითი

<div>ეს არის პალიტრა</div>

დავალება

შექმენით რამდენიმე პალიტრა თქვენს HTML გვერდზე. შეეცადეთ მოაწყოთ სხვადასხვა ელემენტები, რათა გააოგნოთ ვიზუალური ეფექტი.

გაკვეთილი 9: HTML-ის ცხრილები

HTML ცხრილები გამოიყენება მონაცემების ორგანიზებისთვის. ცხრილები საშუალებას გაწვდოს informations-ის ორგანიზება და ჩაწყობა, რაც ხელს უწყობს მათ სურათების და მონაცემების ადვილად გასაგებად წარმოდგენას.

მაგალითი

<table>
    <tr>
        <th>სახელი</th>
        <th>ასაკი</th>
    </tr>
    <tr>
        <td>გიო</td>
        <td>25</td>
    </tr>
</table>

დავალება

შექმენით ცხრილი, რომელიც შეიცავს რამდენიმე რეგისტრირებულ მონაცემს. დაამატეთ უფრო მეტი ხაზი და სვეტი.

გაკვეთილი 10: HTML-ის მრავალფეროვნება

HTML-ის მარკაპი შეიძლება შეიცავდეს უამრავ ელემენტს, რაც თქვენს ვებსაიტს უფრო საინტერესოს და ფუნქციონალურ ხდის. HTML-ის მარკაპი საშუალებას გაწვდოს მომხმარებლებს გადაადგილება თქვენს ვებსაიტზე და მიიღონ საჭირო ინფორმაცია.

მაგალითი

<div>ეს არის მრავალფეროვანი მარკაპი</div>

დავალება

შექმენით თქვენი HTML გვერდი, რომელშიც მრავალფეროვანი ელემენტებია. გაატარე ექსპერიმენტები სხვადასხვა ელემენტების კომბინაციით.

გაკვეთილი 11: HTML-ის სექციის ელემენტები

HTML-ში არსებობს სხვადასხვა სექციის ელემენტები, როგორიცაა <section>, <article>, და <aside>, რომლებიც გამოიყენება შინაარსის კონტექსტual ორგანიზებისთვის. ეს ელემენტები მნიშვნელოვანი ხდის ვებსაიტის გამართულობას და შინაარსის ეფექტურ წარმოსახვას.

მაგალითი

<section>
    <h2>ეს არის სექცია</h2>
    <p>შინაარსი აქ იქნება</p>
</section>

დავალება

შექმენით რამდენიმე სექცია HTML გვერდზე და მოაწყვეთ თქვენი შინაარსი.

გაკვეთილი 12: HTML-ის ნავიგაცია

HTML-ის ნავიგაციის ელემენტებია <nav>, რომელიც გამოიყენება ვებსაიტზე ლინკების ჯგუფირებისათვის. ეს ხელს უწყობს მომხმარებლებს უკეთესად გადაადგილებაში და შეტყობინებების გაცნობაში.

მაგალითი

<nav>
    <ul>
        <li><a href="#home">მთავარი</a></li>
        <li><a href="#about">მარტივად</a></li>
    </ul>
</nav>

დავალება

შექმენით ნავიგაციის მენიუ თქვენს HTML გვერდზე და დაამატეთ ლინკები სხვა სექციებზე.

გაკვეთილი 13: HTML-ის მარკაპი

HTML მარკაპი საშუალებას გაწვდოს შინაარსის ორგანიზება და სტრუქტურა. სხვადასხვა ელემენტების კომბინაცია შექმნის ლამაზ და ფუნქციონალურ ვებსაიტს. HTML-ის მარკაპის სწორად გამოყენება მნიშვნელოვანია ვებსაიტის ოპტიმიზაციისთვის.

მაგალითი

<header>
    <h1>ეს არის ჩემი ვებსაიტი</h1>
</header>

დავალება

შექმენით HTML გვერდი, რომელიც მოიცავს ყველა მარკაპის ელემენტს. დაამატეთ სექციები, პარაგრაფები და ლინკები.

გაკვეთილი 14: HTML-ის ეთიკეტი

HTML-ის ეთიკეტი მოიცავს კოდირების წესებს, რომლებიც უნდა გაითვალისწინოთ თქვენი HTML გვერდების შექმნის დროს. კარგ ეთიკეტს ასახავს კოდის სუფთა და გასაგები სტრუქტურა, რაც ზრდის კოდის გაწვდულობას.

მაგალითი

<h1>გამარჯობა</h1>

დავალება

წარმოიდგინეთ თქვენი HTML კოდის ორგანიზება. შექმენით ჩარჩო, რომელშიც იყენებთ საუკეთესო პრაქტიკას.

გაკვეთილი 15: HTML-ის თვალთახედვა

HTML-ის თვალთახედვა მოიცავს ინფორმაციების ვიზუალური განლაგების შესაძლებლობას. ეს საშუალებას გაწვდოს მომხმარებლებს უფრო მარტივად გასაგები და დალაგებული ვებსაიტები.

მაგალითი

<p>ეს არის ჩემი ვებსაიტის პირველი გვერდი</p>

დავალება

შექმენით HTML გვერდი, რომელშიც თვალთახედვის ელემენტები შეიმუშავეთ.

გაკვეთილი 16: HTML-ის გამოყენება პროგრამირებაში

HTML არის ვებსაიტების მარკაპის ენა, რომელიც მნიშვნელოვან როლს თამაშობს პროგრამირების სფეროში. HTML-ის სწორად გამოყენება პროგრამისტებს სჭირდებათ იმისათვის, რომ მოახდინონ კოდირების ეფექტური და სწორად ორგანიზებული.

მაგალითი

<div>ეს არის HTML-ის პროგრამირების მაგალითი</div>

დავალება

შექმენით HTML გვერდი, რომელიც აერთიანებს სხვადასხვა პროგრამირების ელემენტებს, როგორიცაა JavaScript და CSS.

გაკვეთილი 17: HTML-ის მარკაპის სწორი გამოყენება

HTML მარკაპის სწორად გამოყენება აუცილებელია ვებსაიტების ოპტიმიზაციისთვის და სერვერის სწრაფი რეაგირებისთვის. ეს ასევე ხელს უწყობს მომხმარებლის გამოცდილების გაუმჯობესებას.

მაგალითი

<article>
    <h2>ეს არის სტატიის სათაური</h2>
    <p>სტატიის შინაარსი აქ იქნება</p>
</article>

დავალება

შექმენით HTML გვერდი, რომელიც შეიცავს მარკაპის ელემენტებს და სწორად ორგანიზებულ შინაარსს.

გაკვეთილი 18: HTML-ის გამოყენება SEO-ში

HTML-ის სწორად გამოყენება ეხმარება ვებსაიტების SEO ოპტიმიზაციისთვის. მარკაპის სწორად გამოყენება, როგორიცაა <title>, <meta> და <header>, მნიშვნელოვნად ეხმარება ვებსაიტის სინდუქტივობის გაუმჯობესებას.

მაგალითი

<meta name="description" content="ეს არის ჩემი ვებსაიტი">

დავალება

შექმენით HTML გვერდი, რომელიც შეიცავს SEO ოპტიმიზაციისთვის საჭირო ატრიბუტებს.

გაკვეთილი 19: HTML-ის სექციების სწორი ორგანიზება

HTML-ის სექციების სწორად ორგანიზება საჭიროა ვებსაიტის ვიზუალური დალაგებისა და შინაარსის სასარგებლოდ. სწორად ორგანიზებული HTML სტრუქტურა იმსახურებს მომხმარებლის ყურადღებას.

მაგალითი

<footer>
    <p>© 2024 ჩემი ვებსაიტი</p>
</footer>

დავალება

შექმენით HTML გვერდი, რომელშიც სწორად ორგანიზებული სექციებია, რათა მომხმარებლებს მარტივად შეძლონ ვებსაიტზე გადაადგილება.

გაკვეთილი 20: HTML-ის ტესტირება და შეცდომების გადატანა

HTML-ის კოდის ტესტირება და შეცდომების გადატანა მნიშვნელოვანია იმისათვის, რომ მომხმარებლები მაქსიმალურად კომფორტულად და ეფექტურად შეძლონ ვებსაიტის გამოყენება. HTML-ის შეცდომების მეშვეობით გაწვდილი კოდი უნდა იყოს სუფთა და სწორად ფორმატირებული.

მაგალითი

<html>
    <head>
        <title>გამარჯობა</title>
    </head>
    <body>
        <p>ეს არის HTML-ის ტესტირება</p>
    </body>
</html>

დავალება

შექმენით HTML გვერდი და გამართეთ იგი, რათა გამოავლინოთ რაიმე შეცდომები. ისწავლეთ HTML-ის ტესტირების ტექნიკები და გაიგეთ, როგორ უნდა გადაიტანოთ შეცდომები.
